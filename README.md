### Prerequisites

- Node.js (v14 or later recommended)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/sqrdao-intern/danang-hub-landing.git
   cd danang-hub-landing
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally

Start the development server:

```bash
npm run dev
```

Visit `http://localhost:5173` (or the port shown in your terminal) to view the site.
